# RE:Hear
> **창업 실패 공유담 서비스**</br>
> Re-hear은 창업 실패 경험을 가진 멘토들의 이야기를 다시 새겨 들어 예비 창업자의 Rehearsal을 제공하는 플렛폼입니다.
>
> 2022 동국대학교 창업동아리 서리담</br>
> 프로젝트 기간 : 2022.01.25 ~


</br>
